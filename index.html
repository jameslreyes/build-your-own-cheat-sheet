CSS Padding & Margin Reference

Padding Properties

Property Description
padding: Creates space around an element's content, inside of any defined borders
padding-right: Adjusts padding on the right side of an element
padding-left: Adjusts padding on the left side of an element
padding-top: Adjusts padding on the top of an element
padding-bottom: Adjusts padding on the bottom of an element

Margin Properties

Property Description
margin: Creates space around elements, outside of any defined borders
margin-right: Adjusts space on the external right side of an element
margin-left: Adjusts space on the external left side of an element
margin-top: Adjusts space on the external top of an element
margin-bottom: Adjusts space on the external bottom of an element
